PROJECT TITLE: ArtHTML

PURPOSE OF PROJECT: Convert image to HTML Table

VERSION or DATE: release candidate

HOW TO START THIS PROJECT: ArtHTML.main(String args[])

AUTHORS: Antonio Bianco

USER INSTRUCTIONS:
Avviando il programma con "java -jar arthtml.jar" verrà visualizzata una finestra di dialogo per scegliere l'immagine da coinvertire.

È possibile avviarlo da terminale evitando la visualizzazione dell'interfaccia grafica così:
java -jar arthtml.jar ~/Immagini/Icone/Java.png
